# LeetCode Solutions

This repository contains my solutions to various problems from LeetCode in order to register my progress and showcase my problem solving skills.

Current repo organization is something like this:
- `normal-problems/` - Python (and sometimes C) solutions for common problems;
- `algorithms/` - Classical algorithms and data structures;
- `30-days-of-pandas/` - Pandas exercises to focus on data manipulation;

Each problem lives in its own folder with:
- The solution file(s)
- A `README.md` explaining the reasoning and thought process behind the solution

- Algorithms: 1 implementation
- 30 Days of Pandas: 7 solutions
