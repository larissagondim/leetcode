## Description

Given two integers `num1` and `num2`, return the **sum** of the two integers.

## Thought Process

1. **Core Idea**: This is the most straightforward problem on LeetCode — the solution is literally returning `num1 + num2`.

2. **Step-by-step**:
   - We receive `num1` and `num2`
   - Return `num1 + num2`

3. **Problem Purpose**: It serves as an introduction to the LeetCode platform, so the user can get familiar with the submission interface, input/output types, and LeetCode's class structure.
